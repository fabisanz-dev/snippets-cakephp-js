### Plugin datetimePicker (Bootstrap 3)
    
        https://eonasdan.github.io/bootstrap-datetimepicker/
            ## Requirimientos 
                jQuery: https://cdnjs.com/libraries/jquery/
                Moment.js && Locales: Moment's locale files: https://momentjs.com/
                Bootstrap CSS, Bootstrap.js (transition and collapse are required if you're not using the full Bootstrap) v3+: https://getbootstrap.com/docs/3.3/
                Bootstrap Datepicker script: https://bootstrap-datepicker.readthedocs.io/en/latest/
                
                Bootstrap Datepicker CSS JS: https://eonasdan.github.io/bootstrap-datetimepicker/Installing/#installation-guides
            
            ## Como debe quedar
            
                <link rel="stylesheet" href="/path/to/bootstrap-datetimepicker/build/css/bootstrap-datetimepicker.min.css" />
                
                <script type="text/javascript" src="/path/to/jquery.js"></script>
                <script type="text/javascript" src="/path/to/moment.js"></script>
                <script type="text/javascript" src="/path/to/bootstrap/js/transition.js"></script>
                <script type="text/javascript" src="/path/to/bootstrap/js/collapse.js"></script>
                <script type="text/javascript" src="/path/to/bootstrap/dist/bootstrap.min.js"></script>
                <script type="text/javascript" src="/path/to/bootstrap-datetimepicker.min.js"></script>
                
            ## Usage
            
                # 1: Input con boton de calendario y localizacion: es
                <div class="container">
                    <div class="row">
                        <div class='col-sm-6'>
                            <div class="form-group">
                                <div class='input-group date' id='datetimepicker2'>
                                    <input type='text' class="form-control" />
                                    <span class="input-group-addon">
                                        <span class="glyphicon glyphicon-calendar"></span>
                                    </span>
                                </div>
                            </div>
                        </div>
                        <script type="text/javascript">
                            $(function () {
                                $('#datetimepicker2').datetimepicker({
                                    locale: 'ru'
                                });
                            });
                        </script>
                    </div>
                </div>
                
                # 2 Sin iconos
                    <div class="container">
                    <div class="row">
                        <div class='col-sm-6'>
                            <input type='text' class="form-control" id='datetimepicker4' />
                        </div>
                        <script type="text/javascript">
                            $(function () {
                                $('#datetimepicker4').datetimepicker();
                            });
                        </script>
                    </div>
                </div>
            
            
            ## TIP: para guardar en la base de datos, en caso de que cambie el formato (php - sql): https://tecadmin.net/convert-date-format-in-php/
